Neste projeto houve um foco maior em trabalhar fundamentos do Excel. Para isso, foi utilizada uma base de dados de vendas de bikes.

Trabalhamos fundamentos como a limpeza e formatação dos dados em novas categorias, operações com estes dados, tabelas dinâmicas, confecção de gráficos, segmentação de dados e por fim, a criação de um dashboard interativo.
